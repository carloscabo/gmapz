Responsive y bloqueo del scroll / zoom del mapa

Una de las funcionalidades más interesantes de GMapz es que está preparado para móbiles y tabletas. Cuando se encuentra en un dispositivo mobile el scroll y el zoom se bloquen automáticamente y el usuario decide cuando desbloquearlo con un control personalizado.

Para esta funcionalidad hace falta incluir la librería MQBE.js (Media Query Breakpoint Events), y una hoja de estilo que defina los diferentes breakpoints y el estilo del control de bloqueo que se añade en el mapa, los estilos empleados en este ejemplos pueden encontrarse en gmapz.responsive.css ( ver ).

Sample 5A   Bloqueo automático en responsive

map goes here

Encoge el navegador por debajo de 768 para que el mapa se bloquee automáticamente.

El estilo del control "Scroll enabled / disabled" puede editarse en gmapz.responsive.css. A continuación el js necesario para esta demo...

// Start map with default options
map_sample_5A = new GMapz.map($('#map-sample-5A'));
 
map_sample_5A.onReady = function() {
 
  // Enables responsive control and events
  this.addScrollControl();
 
  // Load Italy cities
  this.addLocations(italy_cities).fitBounds();
 
  // Responsive events take a look at:
  // https://github.com/carloscabo/MQBE
  $(document)
    .on('enter.mobile.mqbe', function() {
      map_sample_5A.lockScroll();
    }).on('leave.mobile.mqbe', function() {
      map_sample_5A.resumeScroll();
    });
 
};

Podemos activar y desactivar el scroll-lock desde JS.

Sample 5B   Mapa inicialmente bloqueado

map goes here

Podemos iniciar el mapa con el scroll bloqueado por defecto, y hacer que se desbloquee únicamente por la interacción del usuario, ya sea pulsando en el botón Activar scroll o al hacer zoom con los controles estandard del mapa.

map_sample_5B = new GMapz.map($('#map-sample-5B'));
 
map_sample_5B.onReady = function() {
  // Load Italy cities
  this.addLocations(italy_cities).fitBounds();
  // Enables responsive control
  this.addScrollControl().lockScroll();
};
