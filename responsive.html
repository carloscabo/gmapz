Responsive

Una de las funcionalidades más interesantes de GMapz es que está preparado para móbiles y tabletas. Cuando se encuentra en un dispositivo mobile el scroll y el zoom se bloquen automáticamente y el usuario decide cuando desbloquearlo con un control personlizado.

Para esta funcionalidad hace falta incluir la librería MQBE.js (Media Query Breakpoint Events), y una hoja de estilo que defina los diferentes breakpoints y el estilo del control de bloqueo que se añade en el mapa, los estilos empleados en este ejemplos pueden encontrarse en gmapz.responsive.css ( ver ).

El pin que se emplea para definir la ubicación actual del usuario se define en gmapz.pins.js con la clave espacial: user_location. A continuación el js necesario para esta demo...

// Start map with default options
map_sample_10 = new GMapz.map($('#map-sample-10'));
 
map_sample_10.onReady = function() {
 
  // Enables responsive control and events
  this.addScrollControl();
 
  // Load Italy cities
  this.addLocations(italy_cities).fitBounds();
 
  // Responsive events
  MQBE.on('enter', 'mobile', function() {
    // Lock map on mobile
    map_sample_10.lockScroll();
  }).on('leave', 'mobile', function() {
    // Restore scroll on desktop
    map_sample_10.resumeScroll();
  });
 
};
